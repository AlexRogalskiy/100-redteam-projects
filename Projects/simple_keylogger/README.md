# keylogger
a simple keylogger in python that send the keystrokes to attacker's server  

<h3>Usage:</h3>  

change the 'SERVER_ADDRESS' to your device's IP and run the server  

```
python3 server.py
```  

and to run the keylogger  

```
python3 keylogger.py
```  

Works well in LAN
